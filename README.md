## Gem3LAB
``
제미나이(Gemini) 3.0으로 만드는 다양하고 이상한 것들 / 링크는 아래와 같은 형태로 입력하시오 ▼
``
```
https://jtech-co.github.io/Gem3LAB/(사이트 제목).html
```
``
순서: 만든 순서 오름차순
``
* **[Aeterna](<https://jtech-co.github.io/Gem3LAB/Aeterna.html>)**: 화려한데 심플하고, 모던한데 클래식한 웹사이트
* **[Aura](<https://jtech-co.github.io/Gem3LAB/Aura.html>)**: 눈에 확 띄는데, 너무 튀지는 않는 웹사이트
* **[Full_void](<https://jtech-co.github.io/Gem3LAB/Full_void.html>)**: 꽉 찼는데, 여백의 미가 느껴지는 웹사이트 
* **[Studio_vibe](<https://jtech-co.github.io/Gem3LAB/Studio_vibe.html>)**: 자유분방한 느낌인데, 정돈된 느낌인 웹사이트
* **[Lumen](<https://jtech-co.github.io/Gem3LAB/Lumen.html>)**: 임팩트 있는데, 은은한 스타일의 웹사이트
* **[Essence](<https://jtech-co.github.io/Gem3LAB/Essence.html>)**: 꾸민 듯 안 꾸민 듯한 웹사이트 
* **[LESSENCE](<https://jtech-co.github.io/Gem3LAB/LESSENCE.html>)**: 내추럴한데 고급스러운 웹사이트
* **[Canvas_zero](<https://jtech-co.github.io/Gem3LAB/Canvas_zero.html>)**: 초등학생도 쓸 수 있을 만큼 직관적인데 전문가가 써도 부족함 없는 디테일한 기능이 숨겨져 있고, 최신 유행하는 힙한 인터랙션을 넣었지만 10년 뒤에 봐도 질리지 않는 타임리스(Timeless)한 디자인이며, 가이드는 친절하게 해주되 사용자를 귀찮게 하는 팝업은 없어야 하고, 손맛이 느껴지는 아날로그 감성이지만 로딩은 0.1초 만에 끝나는 첨단적인 디자인의 웹사이트 (온라인 메모장)
* **[Gama_labs](<https://jtech-co.github.io/Gem3LAB/Gama_labs.html>)**: 공기업처럼 묵직한 신뢰감을 주면서도 스타트업처럼 톡톡 튀는 위트가 느껴지고, 세상에 없던 파격적인 심볼인데 지나가는 할머니가 봐도 우리 회사가 뭐 하는 곳인지 단번에 알아야 하며, 글로벌 시장에 먹히는 세련된 서구적 감각이지만 한국적인 '정(情)'서가 깊게 깔려 있고, 여백이 많아 시원한데 회사의 50년 역사가 그 작은 로고 안에 꽉 차게 담겨 있는 디자인의 웹사이트 (도정 회사)
* **[Jinsim](<https://jtech-co.github.io/Gem3LAB/Jinsim.html>)**: 단 한 줄만 읽어도 눈물이 날 만큼 감성적인데 당장 구매 버튼을 누르게 만드는 이성적인 소구 포인트(CTA)가 확실하고, 우리 브랜드를 모르는 사람도 혹하게 만드는 대중성이 있지만 기존 충성 고객들만 알아듣는 '우리만의 코드'도 들어가야 하며, 점잖은 톤앤매너를 유지하되 SNS에서 밈(Meme)으로 쓰일 만큼 빵 터지는 파급력이 있는 문구가 있는 웹사이트 (물컵 판매)
* **[The_void](<https://jtech-co.github.io/Gem3LAB/The_void.html>)**: 들어오자마자 압도되는 웅장함이 있는데 내 방처럼 아늑해서 하루 종일 있고 싶고, 인스타에 올릴 포토존은 화려하게 꾸미되 전체적으로는 미니멀하고 차분한 갤러리 같아야 하며, 사람들이 바글바글해서 핫플레이스 느낌은 나야 하지만 동선은 겹치지 않아서 프라이빗하고 조용하게 대화를 나눌 수 있는 공간처럼 구성된 웹사이트 (모델하우스)
